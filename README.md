# Event Helper
Last Gama Academy(**xp13**) assignment

## Utilization

### Installing dependencies
1. Access the directory: **/gama**
2. Type the following command on your terminal:
> npm install

### Runing the server on your local machine
1. Access the directory: **/gama**
2. Type the following command on your terminal:
> ng serve
